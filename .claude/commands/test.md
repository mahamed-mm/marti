Run the project's test suite using XcodeBuildMCP.

1. For an Xcode project: `mcp__xcodebuildmcp__test_sim_name_proj`
2. For a Swift Package: `mcp__xcodebuildmcp__swift_package_test`
3. Report results: total passed, failed, skipped.
4. For each failure, show the test name, the assertion that failed, and the file/line. Do not auto-fix.
